Exploring Hidden Worlds

Our world is complex, and so too is the software it's built upon whether we're aware of it or not. Exploring Hidden Worlds is an example of what it takes to build modern software and the complexity of code. This visualization explores Leaflet. An open source JavaScript library for building interactive maps. A popular tool that you may have never noticed before.

The visualization is broken into two parts: First the 3D network of the codebase showing the heirarchy of the project and how different files and pieces of code reference each other. Second, on the right side of the screen, is a dendogram only showing the file structure of the project. Interacting with either manipulate both visualizations, so explore and enjoy!
